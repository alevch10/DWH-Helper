# Описание
Задача модуля установить соединение с S3 хранилищем, а также делать GET, POST, PUT, Patch запросы в него. 

# Задача: 
1) Создай модуль s3
2) Инициализуй в нем библиотеку boto3 с signature_version="s3v4" Для GET операций и "s3" для остальных
3) Создай файл с API: GET, POST, PUT, Patch
4) Подключи роутер в main.py в отдельный тег S3
5) Пробрось переменные в settings в отдельный класс s3
6) Наполни .env заглушками секретов
